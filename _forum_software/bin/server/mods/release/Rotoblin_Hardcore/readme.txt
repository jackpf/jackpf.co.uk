Rotoblin Hardcore by jackpf

Installation:
	Drag and drop to your L4D install directory!
	Note: rename rotoblin.cfg if you want to be able to run normal rotoblin (without hardcore).

Usage:
	Exec rotoblin.cfg to enable rotoblin hardcore mode.

Features:
	No pill spawns outside the saferoom (4 pills only, or 2 for 2v2, 1 for 1v1 etc...).
	No items; no molotov, pipe bomb, gas cans, propane or oxygen.
	Only 1 HR can be picked up.
