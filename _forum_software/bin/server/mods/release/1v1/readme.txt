1v1 by jackpf

Installation:
	Drag and drop to your L4D install directory!

Usage:
	Exec 1v1.cfg to enable 1v1 mode.
	Exec 4v4.cfg to disabled 1v1 mode (the round will need to be restarted).

Features:
	Hunter only.
	Hunters do 33dmg (+ pounce damage) then die.
	Decreased spawn timers, common limit & tank hp.
	Rules are displayed, wallkick is blocked.
